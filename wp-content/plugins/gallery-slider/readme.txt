== Description ==

A quick, easy way to add a slider with captions OR image slider with captions inside Wordpress.

Gallery Slider is a completely free wordpress slider plugin with CSS3 transitions ( with jQuery fallback ) and drag-n-drop UI admin panel.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate Gallery Slider plugin through the 'Plugins' menu in WordPress
3. Plugin can be used via menu item viz. Gallery Slider
